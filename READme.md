# Automatic-Plant-Irrigation-System
As part of the course ‘Logic Circuit Design’, we have built an 
automatic plant irrigation system using NE555 IC and a moisture 
sensor. Using the sensor, we can check whether the soil has adequate 
moisture and if not, supplies water to via a pump/motor, which 
provides it water until it has reached a certain moisture level. When 
people go on vacations or are not present at home for a long period of 
time, they forget to water their plants. This project is an excellent 
solution for such problems, so plants can be watered automatically, 
without any human interference.
